One thing to start

1
Start the local agent backend once (it powers the sidebar; runs on 127.0.0.1:8002):
cd C:\dev\aetherbrowser; ./start.ps1 -BackendOnly
2
Click the AetherBrowser icon in your toolbar to open the side panel. The status above turns green when it connects.

What you get

  • A multi-agent squad in a side panel on any page — ask about This Page, run Research, add another + Spot that shares the same context pool.
  • Governed by default: read-only planning and answering run freely; credential, payment, publish, delete, and other state-changing actions are held for your review.
  • Same backend as the full AetherBrowser desktop app and the iOS app — one brain, many surfaces.
